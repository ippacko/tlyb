# The Luca Yeager Band

A static landing page for **The Luca Yeager Band** (TLYB) with a modern responsive design and lightweight JavaScript enhancements (carousel, animated starfield, smooth menu, and contact quick‑link).

## Getting started

### Run locally
- Open `index.html` directly in your browser (works offline).
- Or use a local dev server (recommended) for best experience:
  - `npx serve` (requires Node.js)
  - `python -m http.server 8000` (Python 3)

## Files
- `index.html` — main landing page
- `style.css` — site styles
- `script.js` — interactive animations and controls
- `logo.png`, `instagram.png`, `spotify.png`, `music.png`, `tiktok.png` — assets
